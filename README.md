## ğŸ§© Modules and Features  

### **1. Donor Management**
- Register donors (restaurants, hostels, event organizers).  
- Store donor details like name, location, and contact information.  
- Maintain donation history for each donor.  

### **2. Food Donation Management**
- Add food details (type, quantity, expiry time).  
- Automate notifications to nearby NGOs and volunteers.  
- Update food status once it is collected or delivered.  

### **3. Volunteer Management**
- Register volunteers with contact and location details.  
- Assign tasks automatically using Salesforce flows.  
- Track pickup and delivery operations.  

### **4. NGO Management**
- Register NGOs and store organization details.  
- Receive notifications of available food.  
- Track the food received and number of people served.  

### **5. Delivery and Tracking**
- Record pickup and delivery details in real time.  
- Update status automatically using record-triggered flows.  
- Display complete delivery reports and analytics.  

### **6. Reports and Dashboards**
- Display total food donations, deliveries, and beneficiaries.  
- Visualize performance using charts and KPIs.  
- Monitor active donors, volunteers, and NGOs.
